
## Group By  

### 정의  

- 데이터를 그룹짓고, 값들을 집계함수로 연산하는 메서드  

### 사용법  

```python
DataFrame.groupby(by=None,
                  axis=<no_default>,
                  level=None,
                  as_index=True,
                  sort=True,
                  group_keys=True,
                  observed=<no_default>,
                  dropna=True)[source]
```
