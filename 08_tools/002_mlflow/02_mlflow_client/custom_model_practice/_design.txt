
(1) DataHandler 클래스
- 필드 : train_data_schea, train_data_path
- 메서드 : load_data, preprocess

(2) Trainer 클래스  
- 필드 : train_parameter, artifacs, metric
- 메서드 : train

(3) Predictor 클래스
- 필드 : signature, artifact_path, dependencies
- 메서드 : predict, load_context(후순위), to_mlflow(후순위)
- Predictor 는 mlflow.pyfunc.PythonModel 를 상속함
- signature 는 mlflow.model.ModelSignature 클래스의 인스턴스

(4) ModelOrchestrator 클래스
- 필드 : data_handler, trainer, predictor
- 메서드 : train_pipeline, predict_pipeline, model_regist_pipeline(후순위), model_deployment_pipeline(후순위)
- data_handler : DataHandler 클래스의 인스턴스
- trainer : Trainer 클래스의 인스턴스
- predictor : Predictor 클래스의 인스턴스
